# Portal Notícias (layout com floats)

Pequeno portal de notícias feito em HTML/CSS, usando **floats** propositalmente (exercício de layout sem flex/grid).

## Páginas
- Home: layout 3 colunas (primário, secundário, lateral)
- Brasil: layout 2 colunas
- Fotos: layout 1 coluna (foco em mídia)
- Nova legislação: página de detalhe (conteúdo principal)
